# Frontend Design Skill

Create distinctive, production-grade frontend interfaces that avoid generic "AI slop" aesthetics. Implement real working code with exceptional attention to aesthetic details and creative choices.

## Design Thinking

Before coding, understand the context and commit to a BOLD aesthetic direction:

- **Purpose**: What problem does this interface solve? Who uses it?
- **Tone**: Pick a clear direction - don't hedge. Options include: brutally minimal, maximalist rich, retro-futuristic, organic/natural, luxury/refined, playful/whimsical, editorial/magazine, brutalist/raw, art deco/geometric, soft/pastel, industrial/utilitarian, warm/inviting, sleek/professional
- **Constraints**: Technical requirements, accessibility needs, existing design system
- **Differentiation**: What makes this MEMORABLE? What's the one thing someone will remember?

**CRITICAL**: Choose a clear conceptual direction and execute it with precision. Bold maximalism and refined minimalism both work - the key is intentionality, not intensity.

## Frontend Aesthetics Guidelines

### Typography

Choose fonts that are beautiful, unique, and interesting. **AVOID** generic fonts like Arial, Inter, Roboto, and system fonts. Instead:

- Use distinctive display fonts paired with refined body fonts
- Consider the emotional weight of font choices
- Vary between serif and sans-serif based on context
- Pay attention to line-height, letter-spacing, and font-weight
- Use typographic hierarchy to create clear visual flow

### Color & Theme

Commit to a cohesive palette. Use CSS variables for consistency.

- Dominant colors with sharp accents outperform timid, evenly-distributed palettes
- Don't be afraid of dark themes, bold colors, or unexpected combinations
- Consider the emotional resonance of color choices
- Ensure sufficient contrast for accessibility

### Motion & Animation

Use animations for delight and feedback:

- Prefer CSS-only solutions when possible
- Focus on high-impact moments: page load reveals with staggered `animation-delay`
- Use scroll-triggered animations sparingly but effectively
- Hover states that surprise and delight
- Micro-interactions that feel responsive

### Spatial Composition

Break free from predictable layouts:

- Unexpected grid arrangements
- Asymmetry with purpose
- Overlap and layering for depth
- Diagonal flow to guide the eye
- Grid-breaking elements for emphasis
- Generous negative space OR controlled density (commit to one)

### Backgrounds & Visual Details

Create atmosphere and depth:

- Gradient meshes and blurs
- Noise textures and grain overlays
- Geometric patterns
- Layered transparencies
- Dramatic shadows
- Decorative borders and dividers
- Custom cursors where appropriate

## What to NEVER Do

Avoid generic AI-generated aesthetics:

- Overused font families (Inter, Roboto, Arial, system fonts)
- Cliché color schemes (particularly purple gradients on white backgrounds)
- Predictable card-based layouts with uniform spacing
- Cookie-cutter design that lacks context-specific character
- The same design every time - VARY between light/dark, different fonts, different aesthetics
- Converging on common "safe" choices across generations

## Implementation Guidance

Match implementation complexity to the aesthetic vision:

- **Maximalist designs** need elaborate code with extensive animations, effects, and details
- **Minimalist designs** need restraint, precision, and careful attention to spacing, typography, and subtle refinements
- Elegance comes from executing the vision fully, not from complexity alone

## Remember

Claude is capable of extraordinary creative work. Don't hold back. Show what can truly be created when thinking outside the box and committing fully to a distinctive vision.

Interpret creatively. Make unexpected choices. Design something genuinely memorable for the specific context - not a generic template that could be used anywhere.
