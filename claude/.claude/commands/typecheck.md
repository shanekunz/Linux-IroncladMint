description: Typecheck and fix errors
---

Run `pnpm typecheck` and fix any TypeScript errors.

Instructions:

1. Run the typecheck command first to see all errors
2. Fix each error at its root cause - don't just add type assertions or `any`
3. If a fix requires understanding context, read the relevant files first
4. After fixing, run typecheck again to verify no new errors were introduced
5. Summarize what you fixed and why
